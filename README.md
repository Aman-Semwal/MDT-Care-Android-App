# MDT Care - Mobile Diagnostic Tool for Android

## Description
MDT Care is an Android application developed using Java in Android Studio. It helps users analyze device information, battery status, network connectivity, sensor functionality, storage details, and generate a final diagnostic report.

## Features
- Device Information (Model, Brand, Android Version)
- Battery & Network Status
- Sensor Testing (Accelerometer, Gyroscope, etc.)
- Storage & Memory Analysis
- Final Diagnostic Report

## Tech Stack
- Java
- Android Studio
- XML UI Design
- Android APIs (BatteryManager, SensorManager, ConnectivityManager)

## Project Structure
- MainActivity.java
- DeviceInfoActivity.java
- BatteryNetworkActivity.java
- SensorTestActivity.java
- StorageActivity.java
- ReportActivity.java

## How to Run
1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on emulator or real device
